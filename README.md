# Open Invitation

Open Invitation is a podcast highlighting open access efforts from across the University of Idaho.

<https://cdil.lib.uidaho.edu/open-invitation/>

This website project uses a static web approach to host and promote a podcast series. 
It is built on [Oral History as Data](https://github.com/oralhistoryasdata) and [CollectionBuilder](https://collectionbuilder.github.io/) template, with customizations to support serialization to external podcast platforms.
